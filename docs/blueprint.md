# **App Name**: KP Commute

## Core Features:

- Real-Time Tracking: Display live locations of available buses and cabs in real-time on a map using the Google Maps API.
- Crowd Prediction: Predicts crowd levels on buses using an LSTM-GNN AI model, displaying color-coded indicators (Green/Yellow/Red).
- AI Explanations: Uses Explainable AI tool popups to provide reasoning behind delays, like 'Bus 73 delayed by 8 mins (Exam rush hour impact)'.
- Point System: Enables users to track rides and report crowd levels to earn points.
- Gamified Rewards: Allows users to earn points for actions such as tracking rides, ride-sharing, reporting crowd levels, and referring friends.
- Reward Redemption: Redeem points for rewards like free ride vouchers and gift cards.
- Smart Trip Planner: Display a smart trip planner optimizing routes by suggesting the best combination of transport, considering cost and time.

## Style Guidelines:

- Primary color: Saffron (#FF9933) to reflect the Indian heritage.
- Background color: Off-white (#F0F0F0) to provide a clean and neutral base.
- Accent color: Green (#3CB371) to symbolize progress and positive actions.
- Body and headline font: 'PT Sans' (sans-serif) for clear readability and a modern feel.
- Use illustrations of auto-rickshaws, buses, and trains as icons to enhance visual communication.
- Incorporate a progress bar to indicate points accumulation towards the next reward.
- Use subtle animations for real-time updates on the map and point earnings.
